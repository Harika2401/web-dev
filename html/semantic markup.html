tags are of 2 types

  • Sematic tags
  • non-sematic tags

Semantic tags specifies the meaning of tags or we can understand the meaning of tags just by seeing them

for example paragraph tag and heading tag

Non-Semantic tags doesn't specifies the meaning of tags or we can't understand the meaning of tags just by seeing them

for example div tag and span tag

we mostly use semantic tags for 2 reasons

  • proper meaning
  • SEO friendly(Search Engine optimization) if we search for anything then chrome or anything searches for headers if you use div instead of header then your website may be not visible properly
  • Make it readable , normally websites uses screen readers so to help these screen readers this semantic tags should be used
